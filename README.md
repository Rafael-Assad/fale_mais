# A Telzir apresenta seu novo produto, Fale+

O objetivo desse projeto é demonstrar para o cliente quais são as vantagens de contratar o novo produto da Telzir, para aquelas pessoas que precisar ficar em contato com pessoas de toda São Paulo. 

A página inclui um simulador, para o cliente saber quanto vai economizar por ligação e ainda uma página super facilitada para assinar o plano que desejar 

## página de Simulação

Essa página é especialmente configurada para que o cliente possa entender como funciona a tarifa num plano regular da Telzir e como uma ligação fica mais em conta com um dos planos da Fale+

### `Tecnologias usadas`

O projeto é feito em JavaScript, com uso de React, para melhor manipulação de componentes.

dependencias usadas:
 - Styled Components;
 - React-Router-Dom;
 - React-Hook-Form;
 - Yup + Resolvers;
 - React-Icons

>Também foi adicionado Axios, mas apenas como uma desmonstração de como ele ficaria, caso um backend já estivesse pronto pra ser usado através de una API
